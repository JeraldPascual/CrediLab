# Components

Shared UI components used across pages.

| Component | Purpose |
|---|---|
| `Sidebar.jsx` | Main navigation sidebar with route links, theme toggle, logout |
| `StudentHeader.jsx` | Top bar with credits display, wallet status, notifications |
| `ProtectedRoute.jsx` | Auth gate — redirects unauthenticated users to login |
| `PublicRoute.jsx` | Inverse gate — redirects authenticated users to dashboard |
