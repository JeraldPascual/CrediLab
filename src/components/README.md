# Components

Shared UI components used across pages.

| Component | Purpose |
|---|---|
| `Sidebar.jsx` | Main navigation sidebar with route links, theme toggle, logout |
| `StudentHeader.jsx` | Top bar with credits display, wallet status, notifications |
| `CLBBalanceStrip.jsx` | Persistent credit balance + pool remaining strip below header |
| `ProtectedRoute.jsx` | Auth gate — redirects unauthenticated users to login |
| `PublicRoute.jsx` | Inverse gate — redirects authenticated users to dashboard |
| `MobileWarningBanner.jsx` | Dismissible banner warning mobile users about optimal desktop experience |
| `ui/StickyScrollReveal.jsx` | Aceternity-style sticky scroll reveal for landing page features |
| `ui/CanvasText.jsx` | Animated canvas text with colorful horizontal band effect |
